CSV Upload & Validation Backend
This project is a simple backend service built using FastAPI that allows users to upload CSV files, validate their contents, and fetch processed records. The focus of this project is safe file handling, data validation, proper logging, and clean API design.

This task was implemented as part of a backend internship assignment.


# To run locally: 
First create a virtual environment using these commands:
1. pip install virtaulenv
2.python -m virtualenv venv
 
Then install the dependencies using: 

pip install -r requirements.txt
